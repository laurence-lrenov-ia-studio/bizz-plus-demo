MODE ANIMATEUR

Bienvenue aux commandes

Vous êtes connecté en animateur du club. C'est vous qui pilotez la réunion en direct depuis le Cockpit. Voici les trois leviers.

Pointer les présences
Cochez présent ou excusé à l'arrivée — les compteurs se mettent à jour partout, y compris à l'écran.
Désigner les Star(s) du jour
1 à 3 membres mis à l'honneur, qui présentent leur activité devant le réseau.
Animer le tour & projeter
Lancez le tour de présentation chronométré, ou basculez sur grand écran avec le QR visiteurs.
Le badge « Mode animateur » en haut reste cliquable à tout moment.
MODE ANIMATEUR ACTIF

Vous pilotez le club

Le mode animateur (admin) débloque le Cockpit, le tour de présentation et le mode projection. Les membres connectés avec leur code personnel, eux, voient l'annuaire et le programme sans ces commandes.

Ce que vous débloquez
Cockpit de pilotage, pointage des présences, stars du jour, tour chronométré, projection grand écran.
Quitter le mode animateur
Déconnectez-vous pour revenir à l'écran d'accès, puis reconnectez-vous avec un code membre ou visiteur.
